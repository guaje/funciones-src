## Procedimientos
## Procedimientos: Ejemplo en C++ ```cpp #include <iostream> using namespace std; void imprimir(bool x, char y, double z) { cout << "El valor de la variable x es: " << x << endl; cout << "El valor de la variable y es: " << y << endl; cout << "El valor de la variable z es: " << z << endl; cout << endl; } int main() { imprimir(false, '.', 2.8); imprimir(2.8, ',', 5.6); return 0; } ```
## Procedimientos: Ejemplo en Python ```python def imprimir(x, y, z): print("El valor de la variable x es: " + str(bool(x))) print("El valor de la variable y es: " + y) print("El valor de la variable z es: " + str(float(z))) print() imprimir(False, '.', 2.8) imprimir(2.8, ',', 5.6) ```
## Procedimientos: Consideraciones Un procedimiento se puede asimilar a una función que no retorna un resultado, y que puede retornar más de un valor mediante el uso de __argumentos por referencia__.
