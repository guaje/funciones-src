## Funciones en Programación

Funciones en Programación: Definición

Una función, desde el punto de vista de la programación, se define como un proceso que recibe valores de entrada (llamados argumentos) y el retorna un valor resultado.

De la definición matemática: $$f : A \rightarrow B$$

se tiene que \(f\) corresponde al nombre de la función, el conjunto \(A\) corresponde al tipo de los argumentos de dicha función y el conjunto \(B\) que es el rango corresponderá al valor de retorno de dicha función.

## Funciones en Programación: C++ ```cpp #include <iostream> using namespace std; // Espacio para declarar funciones bool esPar(int x) { return x % 2 == 0; } int main() { return 0; } ```
## Funciones en Programación: Python ```python # Espacio para declarar funciones def esPar(x): return x % 2 == 0 ```

Funciones en Programación: Ejemplo

Se definirá una función que eleve un número al cubo. Para expresar una función que calculé esta operación, en primera instancia se construye la expresión \(f : \mathbb{R} \rightarrow \mathbb{R}\) que define que la función tiene como entrada (dominio) un número real y como salida (rango) un número real. La declaración de la función quedará de la siguiente forma: $$f: \mathbb{R} \rightarrow \mathbb{R}$$ $$f: x 􏰀\rightarrow x^2$$

## Funciones en Programación: Ejemplo Definir una función que calcule la n-ésima potencia de 2.
## Funciones en Programación: Ejemplo en C++ ```cpp #include <iostream> using namespace std; int potenciaDe2(int x) { return 1 << x; } int main() { int x = 0; int y = 0; cout << "Ingrese un número: \t"; cin >> x; y = potenciaDe2(x); cout << "La " << x << "-esima potencia de 2 es: " << y << endl; return 0; } ```
## Funciones en Programación: Ejemplo en Python ```python def potenciaDe2(x): return 1 << x x = int(input("Ingrese un número: \t")) y = potenciaDe2(x) print("La " + str(x) + "-esima potencia de 2 es: " + str(y)) ```
## Funciones en Programación: Ejemplo Usando las funciones seno y coseno calcule la tangente.
## Funciones en Programación: Ejemplo en C++ ```cpp #include <iostream> #include <math.h> using namespace std; double tangente(double x) { double s = sin(x); double c = cos(x); return s / c; } int main() { int x = 0; int y = 0; cout << "Ingrese un número: \t"; cin >> x; y = tangente(x); cout << "La tangente de " << x << " es: " << y << endl; return 0; } ```
## Funciones en Programación: Ejemplo en Python ```python import numpy as np def tangente(x): s = np.sin(x) c = np.cos(x) return s / c x = float(input("Ingrese un número: \t")) y = tangente(x) print("La tangente de " + str(x) + " es: " + str(y)) ```

Funciones en Programación: Anotaciones

  • Una forma de entender/ver las funciones es como sub-programas que se pueden invocar desde cualquier parte del programa cuantas veces sea necesario.

    E.g., desde otra función(es) como la función main* o incluso, desde la misma función.

  • Generalmente se usan cuando existen dos o más porciones de algoritmo dentro de un programa que son iguales o muy similares.

* En python no es necesario declarar una función main.
